# Blog frontend project

Implementing a blog front-end using Next.js.

## Development Environment Setup

### Copy .env

```shell
cp .env.example .env
```

### Install dependencies

```shell
pnpm install
```

### Start development server

```shell
pnpm dev
```

### Start Storybook

```shell
pnpm sb
```
